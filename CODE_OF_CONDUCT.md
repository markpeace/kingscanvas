# Code of Conduct

We maintain a respectful, professional environment for all contributors (human and AI).

## Standards
- Be clear and constructive in commits, PRs, and reviews.
- Prefer facts, examples, and references in discussions.
- Prioritize safety: avoid committing secrets or PII.

## Unacceptable Behavior
- Harassment or personal attacks.
- Knowingly introducing malicious code or unsafe configurations.

## Reporting
Report concerns directly to the repository owner: @YOUR_HANDLE

## Enforcement
The owner may restrict access, revert contributions, or archive discussions that violate this code.
