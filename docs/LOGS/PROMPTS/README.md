# Prompt Logs

A log is written **after every prompt** is executed.  
Front-matter example (YAML):

id: prompt-0001
epoch: epoch-0001-bootstrap
issued_at: 2025-10-02T12:00:00Z
by: frontend-ai
intent: Seed CONTRACT and PROTOCOL docs
inputs: []
plan: []
success_criteria: []
side_effects: []
links: {}

Naming: `YYYY-MM-DDTHHMMSSZ-prompt-####.md`
