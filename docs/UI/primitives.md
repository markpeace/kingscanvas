# UI Primitives

This template ships with a curated set of primitives in `components/ui/`:
Button, Input, Textarea, Label, Card, Separator, Tooltip, Dialog, Tabs, Checkbox, Switch, Select.

## Usage
    import { Button, Input, Card, CardHeader, CardTitle, CardContent } from "@/components/ui"

All components are theme-aware (light/dark) and unopinionated; extend via `className`.

## Notes
- Keyboard accessible defaults via Radix.
- Keep new exports centralized in `components/ui/index.ts`.
