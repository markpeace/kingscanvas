# Recording Progress — Checklist (After Every Prompt)

1. Write prompt log under `/docs/LOGS/PROMPTS/`.
2. Update `/docs/STATE/CURRENT.yaml` (active prompt → null if done; next_actions updated).
3. Update epoch `STATUS.yaml`.
4. Open/update PR and write `/docs/LOGS/PRS/` entry.
5. Run local validation scripts and note results in the prompt log.
